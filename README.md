# Dependencies
- Front-end: React
- Back-end: Flask
- Database: Postgresql
- External API: https://type.fit/api/quotes

# Features
- Authenticating a user login and signup.
- A random quote appears when the page renders.

# Environment Set up
- Fork and clone this repo.
- Run `yarn install`
- Run `yarn start`
